# DermaScan AI - Skin Analysis System

AI-powered skin analysis web application built using MERN Stack, Gemini API, Google Maps API, OAuth, and MongoDB Atlas.

---

## Features

- AI Skin Analysis using Gemini API
- Upload Face Image for Analysis
- Diagnosis, Confidence, Severity Detection
- Refer to Dermatologist Suggestion
- Clinical Notes Output
- Google Login (OAuth)
- Doctor Recommendation with Maps
- Analysis History
- Secure MongoDB Atlas Storage

---

## Tech Stack

- MongoDB Atlas  
- Express.js  
- React.js  
- Node.js  
- Gemini API  
- Google Maps API  
- OAuth 2.0
